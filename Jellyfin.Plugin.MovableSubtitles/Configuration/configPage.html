Movable and Sizable Subtitles

Adds an "Aa ⇅" button to the video player. Clicking it opens a floating control panel with arrow buttons to move the subtitles, size controls, quick presets (Top / Middle / Bottom) and a Reset All button.

Keyboard shortcut: Ctrl + Shift + S.

When disabled, no script is served to the web client.
Shows the "Aa ⇅" toggle button during playback. This is the primary way to move/resize subtitles.

Advanced: direct manipulation

Off by default. Enable if you want to drag / scroll directly on the subtitle text instead of using the panel buttons.

Defaults

Starting subtitle size as a percentage of the web client default (50-300).
